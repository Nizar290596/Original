# Python Library mmcFoam

Python functions for post processing of some mmc statistics.
Provides a function to load the eulerianStatistics data into python
for post-processing. 

## Installation

In the python/ folder of mmcFoam execute following commands

```bash
python3 setup.py bdist_wheel
pip install dist/mmcFoamReader-1.0.0-py3-none-any.whl 
```

## Usage

After installing the python library import it, e.g., for eulerianStatisticsData
with:

```python
from mmcFoamReader import eulerianStatisticsData

# Load an eulerian statistics file 
data = eulerianStatisticsData("mmcStatistics/eulerianStatistics/1.0/var")

# Data file to store values() and pos() as np.arrays
# Can plot data over a line with the plotAlongLine() function
```

For reading the particle statistics data see the particleStatisticsData 
class in [particleStatisticsData.py](./mmcFoamReader/particleStatisticsData.py)

### Example

```python
from mmcFoamReader import eulerianStatisticsData
from mmcFoamReader import particleStatisticsData


# =============================================================================
#                           Load Data
# =============================================================================

# Load an eulerian statistics file 
eData = eulerianStatisticsData("mmcStatistics/eulerianStatistics/1.0/var")

# Load the particle statistics - Note: loads all time instances
pData = particleStatisticsData("mmcStatistics/particleSampling/","probe1")

# =============================================================================
#                        Plot Eulerian Data
# =============================================================================
# Plot the eulerian data along a line
# Line defined from point p1 to point p2 in 2D coordinates of [axial,radial]

# Plots along the radial direction
p1 = [10,0]
p2 = [10,10]

x,v = eData.plotAlongLine(p1,p2)

import matplotlib.pyplot as plt

fig = plt.figure()
plt.plot(x,v)

# =============================================================================
#                        Plot Particle Data 
# =============================================================================
# Get the conditioned temperature field on the cond. variable z
# and weighing the particle data with their weight stored in variable wt

binCenter, TCond, TStdDev = pData.conditionalStatistics("z","T",weight="wt")

fig = plt.figure
plt.plot(binCenter,TCond)

```

## Visualize Sub-Volume in ParaView

To visualize the sub-volumes generated by the utility `writeSubVolumeSts` in 
ParaView a script is provided.

Excute following script with the paraview tool in python, `pvpython`.

 ```python
#import paraview
from mmcFoamReader import visualizeSubVolumeInParaView

# If explode view is true, than the sub volumes will be radially distributed
# around the domain to create something similar to an explosion view.
# For further details about the options look at the code documentation
visualizeSubVolumeInParaView('/path/to/case',r=0.01,explodeView=True)
 ```
