# Artificial thickening of the flame support 

This folder contains files supporting Artificial Thickening of the Flame (ATF) approach. ATF is used in Large Eddy Simulations (LES) to create thickened flame on the Eulerian field. In context of MMC, it helps to provide a well-resolved reference field for the stochastic particles.
