## Commit Notations

Following flags can be used to show the intent of commits:

|Flag          | Description                 |
|:-------------|-----------------------------|
| [Fix]        | Fix a bug in the code       |
| [ADD]        | Add an additional functionality or tutorial |
| [CLEANUP]    | If the behavior of the code is not changed and only changes to improve readability are introduced |
| [MOD]        | Modification of an existing library |
